### GreenBuild AI &nbsp;·&nbsp; [GitHub](https://github.com/qasim-you/greenbuild.ai) &nbsp;·&nbsp; [Live](https://greenbuild-ai.vercel.app)

### Nexus — Multi-Agent Network & Energy Optimization &nbsp;·&nbsp; [GitHub](https://github.com/Synthapse/Nexus)

AI-driven system for decentralized network and energy optimization across macro, medium, and micro scales. Autonomous agents communicate peer-to-peer: PSO agents optimize WiFi tower placement, ACO agents find lowest-latency routing paths, and a GNN coordinates real-time load balancing across the full network. Deployed on Google Cloud with Kafka for event streaming and Grafana for live infrastructure monitoring. Node-level behavior validated through ns3 simulations.

`Python` `GNN` `PSO` `ACO` `Apache Kafka` `Grafana` `Google Cloud Run` `Google Compute Engine` `ns3`
